mark-p-thomas : Show Clean Previews

Created by Mark Thomas, last modified on Mar 21, 2024

Rather than changing a track and then undoing the operation, the user may wish to see a preview of the result of the operation. This can be set for global behavior. Currently destructive operations that cannot be tracked with history (undo/redo) always work through previews since the user cannot undo the operations.
