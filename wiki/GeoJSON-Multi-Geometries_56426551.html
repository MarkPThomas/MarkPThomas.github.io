mark-p-thomas : GeoJSON Multi-Geometries

Created by Mark Thomas, last modified on Nov 08, 2023

Multi-geometries are all immutable CoordinateContainer Geometries. These geometries have multiple geometries of the same type. That type is constrained by the class type defined. These constraining classes are described in the child pages.
