mark-p-thomas : Libraries

Created by Mark Thomas, last modified on Nov 03, 2023

Different independent supported libraries are covered in the child topics here. Some of these may be spun off as independent GitHub projects and NPM dependencies. These libraries are used in the apps described in this wiki.

My main Mark.ly project that contains these apps is in a multi-project single repo structure, with the idea that I may build it out as a monolith-to-microservices hybrid. Models for independent apps, and common directories may become large, complex, & independently useful enough to spin off as separate libraries. That is what these are.
