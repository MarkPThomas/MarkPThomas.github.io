mark-p-thomas : Recording Includes Time Not in Activity

Created by Mark Thomas, last modified on Mar 20, 2024

When a GPS device is turned on & off, such as when charging, loading tracks or underlays, etc., some devices immediately start recording and merge these recordings with a later intended recording as one Track. You probably want these erroneous recordings removed.

For example, this loaded track was from a summer trip that I took to Peru in 2024. I had turned on the Garmin GPS in Lima, Peru, in order to do some work with the GPS unit. Since I didn’t work out the unit behavior precisely in order to only start recording a new track once we started hiking, the unit merged both times into the same track, leaving ‘cruft’ in the file. The app easily cleans this cruft from the GPS Track.
