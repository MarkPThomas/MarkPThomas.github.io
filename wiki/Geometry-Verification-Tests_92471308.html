mark-p-thomas : Geometry Verification Tests

Created by Mark Thomas, last modified on Jan 16, 2024

Many aspects of the library involve calculations that must be verified in various forms. This is not just done once on the implementation stage. Instead, verification tests are run at a variety of testing levels, such as unit tests. These calculations are usually done by hand using a spreadsheet or some other automation such that updates, changes, experiments & derivations can easily be done.
